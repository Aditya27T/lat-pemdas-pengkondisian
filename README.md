# lat-pemdas-pengkondisian

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/Aditya27T/lat-pemdas-pengkondisian.git
~~~

Go to the project directory  

~~~bash  
  cd lat-pemdas-pengkondisian
~~~

Start the cpp file  

~~~bash  
  g++ <filename> -o <output name>
~~~

## Usage/Examples

~~~bash
  g++ hello.cpp -o hai
~~~  

~~~bash
  ./hai
~~~
